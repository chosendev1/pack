Images

Responsive images

Add img-fluid class to an img element to get responsive image.

Responsive raised images

Add img-fluid class and img-raised to an img element to get responsive image.

Image thumbnails

In addition to our border-radius utilities, you can use img-thumbnail to give an image a rounded 1px border appearance.

Rounded images

Add rounded class to an img element for getting a rounded image.

Rounded circle images

Add rounded-circle class to an img element to get a rounded circle image.

Raised images

Add img-raised class to an img element to get raised image.
